# Automated Poultry Pen Temperature Control


> COMPONENTS:
  * RASPBERRY PI 3 MOBEL B V1.2
  * DHT11 TEMP SENSOR
  * 28BYJ-48 STEPPER MOTOR
  * ULN2003 DRIVER BOARD
  * MLX90640BAA INFRARED CAMERA

---

| GND          | GND         |
